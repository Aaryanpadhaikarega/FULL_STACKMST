# 🚀 FSD-2 Project (Experiment 16)

A full stack web application with frontend and backend, integrated with automated testing and CI using GitHub Actions.

## 📂 Project Structure
FSD-2/
└── experiment-16/
    ├── frontend/   # React + Vite + Vitest
    └── backend/    # Node.js backend

## 🛠️ Tech Stack
- Frontend: React, Vite  
- Testing: Vitest, Testing Library  
- Backend: Node.js  
- CI/CD: GitHub Actions  

## ⚙️ Setup Instructions

### Clone the repository
git clone https://github.com/your-username/FSD-2.git  
cd FSD-2  

### Run Frontend
cd experiment-16/frontend  
npm install  
npm run dev  

### Run Tests
npm run test  

### Run Backend
cd ../backend  
npm install  
node index.js  

## ✅ Features
- Input form with real-time updates  
- Unit testing using Vitest  
- Automated testing on every push using GitHub Actions  
- Modular project structure  

## 🔄 CI/CD Pipeline
GitHub Actions automatically:
- Installs dependencies  
- Runs tests on every push  
- Ensures code quality
